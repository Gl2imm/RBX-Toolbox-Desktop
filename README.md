### Eport CSV Files:
Exporting CSV files with all the items in your Group or uploaded as your own.
- Clickable Item name that lead to the item page
- Favourites Count
- Current price
- Current sales
- Item ID
- Asset Type
- Calculated 30% earning
- Calculated DevEx based on current rate
